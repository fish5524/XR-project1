# XR-project1

## Meta Simulator

### 選單 Edit > Project Settings

- 找到 XR Plug-in Management 並展開  
    - 確保 OpenXR 已被勾選  
- 找到 OpenXR 並展開  
    - Enabled Interaction Profiles 區塊下加入 Oculus Touch Controller Profile

### 啟動 MetaXRSimulator

- 打開 MetaXRSimulator  
- 將左上角設為 Activate  
- 點擊 Unity 中的 Play 啟動遊戲

### 操縱方式

Keyboard

- WASD: 前後左右
- B: 跳躍

Mouse

- 右鍵+移動: 轉動視角
